如何量測 Sag
L1 自由長 — 車輛懸空，避震器完全伸張時量測
L2 靜態長 — 車停地面，無人騎乘時量測
L3 騎乘長 — 騎士穿戴完整裝備坐上車，腳踏地輔助平衡
Rider Sag = L1 − L3
Static Sag = L1 − L2
Static Sag 建議為 Rider Sag 的 30–40%

預載調整口訣
Sag 太大（避震太軟）→ 增加預載（順時針鎖緊）
Sag 太小（避震太硬）→ 減少預載（逆時針放鬆）
每轉一圈預載約改變 Sag 1–3mm，依車款彈簧而異
阻尼調整順序
1. 先確定 Sag（預載）正確
2. 調整低速壓縮阻尼（影響入彎側傾）
3. 調整回彈阻尼（影響出彎穩定性）
4. 最後微調高速壓縮（若有此功能）
